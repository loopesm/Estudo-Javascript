Introdução ao JavaScript

Podemos fazer a inclusão direto no arquivos HTML ou em um arquivo externo dentro da pasta do projeto.

Caso o script seja extenso, é recomendado que seja feito um arquivo externo.

Podemos incluir o script na Tag HEAD ( executa antes dos código no Body ) ou na Tag BODY ( executa de acordo com com a árvore do código Body ), porém dependendo da usabilidade do script vai ocorrer um erro de interpretação do código.

PRECEDÊNCIA DO CÓDIGO: para interação com elementos HTML devemos incluir o script após o elemento em questão ter sido carregado pela Tag Body dentro do Browser.

COMENTÁRIOS: // = Comentário de uma linha
/* */ = Comentário de multiplas linhas

Variáveis

As Variáveis armazenam determinadas informações.

TIPOS DE VARIÁVEIS:

String: Armazenamento de Textos.

Number: Valores numéricos inteiros ou fracionados, positivos e negativos.

Boolean: Armazenam estados, como TRUE ou FALSE.

REGRAS DE NOMENCLATURA:

Não iniciar com números, somente letras.

Não pode contar caracteres especiais.

Não pode ser iguais as palavras reservadas da linguagem.

Há a diferencição de caracteres Maiuscúlos e Minúsculos nos nomes.

DECLARAÇÃO DE VARIÁVEL:

Utiliza-se a instrução "VAR".

VENDO O VALOR DE UMA VARIÁVEL:

alert(): Abre uma janela com a mensagem.

document.write(): Escreve o valor da váriavel em um elemento dentro do DOM.

console.log(): Escreve o valor da variável no console do navegador.
