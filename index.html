Introdução ao JavaScript

Podemos fazer a inclusão direto no arquivos HTML ou em um arquivo externo dentro da pasta do projeto.

Caso o script seja extenso, é recomendado que seja feito um arquivo externo.

Podemos incluir o script na Tag HEAD ( executa antes dos código no Body ) ou na Tag BODY ( executa de acordo com com a árvore do código Body ), porém dependendo da usabilidade do script vai ocorrer um erro de interpretação do código.

PRECEDÊNCIA DO CÓDIGO: para interação com elementos HTML devemos incluir o script após o elemento em questão ter sido carregado pela Tag Body dentro do Browser.

COMENTÁRIOS: // = Comentário de uma linha
/* */ = Comentário de multiplas linhas

Variáveis

As Variáveis armazenam determinadas informações.

TIPOS DE DADOS PRIMITIVOS ( VALORES IMUTÁVEIS = Não mudam ):

String: Armazenamento de Textos.

Number: Valores numéricos inteiros ou fracionados, positivos e negativos.

Boolean: Armazenam estados, como TRUE ou FALSE.

Undefined: Não definido na memória ( Sem valor ).

NULL (bigint,symbol) Especificado como sem valor na memória ( Sem valor ).

TIPOS DE DADOS POR REFERÊNCIA ( VALORES MUTÁVEIS = Alteram seu valor ):

Array: Armazena Lista de Dados.

Objetos: Armazena Dados.

Function: Executa Ações.

REGRAS DE NOMENCLATURA:

Não iniciar com números, somente letras.

Não pode contar caracteres especiais.

Não pode ser iguais as palavras reservadas da linguagem.

Há a diferencição de caracteres Maiuscúlos e Minúsculos nos nomes.

DECLARAÇÃO DE VARIÁVEL:

Utiliza-se a instrução "LET" ( Mais indicado ).

Utiliza-se a instrução "VAR".

A diferença entre as duas formas de declaração é que com VAR as variáveis podem sem redeclarada sem gerar erro no código. Porém com LET isso não é possível.

DECLARAÇÃO DE CONSTANTES:

Utiliza-se a instrução "CONST".

VENDO O VALOR DE UMA VARIÁVEL E CONSTANTE:

alert(): Abre uma janela com a mensagem.

document.innerHTML(): Escreve o valor da váriavel em um elemento dentro Documento HTML.

document.write(): Escreve o valor da váriavel em um elemento dentro do DOM.

console.log(): Escreve o valor da variável no console do navegador.

Valores da Variáveis Atribuidas:

Variáveis que não tem valor declarado tem um valor automaatico atribuido como "undefined", ou seja, não aponta para nenhum local na memória.

Variáveis podem ter o valor definido pelo código como "null", ou seja, não aponta para nenhum local na memória. Isso é usado quando para a manipulação de dados.

Não podemos declarar a mesma variável mais de uma vez no código.

Não podemos declarar uma variável sem usar LET ou CONST.

FUNÇÃO PARA SABER O TIPO DA VARIÁVEL: " typeof(variavel) "

  • Sinal de Decremento ( -- )

    Pré atribuido ou Pós Atribuido

  • Sinal de Subtração ( - )

  • Sinal de Raiz Quadrada ( ** 0.5 )

  • Sinal de Concatenação de Textos ( + )

Maneira 1: Função " parseInt("string") "

Maneira 2: Função " parseFloat("string") "

Funções do Navegador

OBS.: Todas as Funções podem ou não retornar um valor para o sistema. Caso elas não retornem valor, será atribuido automaticamente o valor "undefined".

Função "alert()": Emite um alerta simples no navegador.

Função "confirm()": Emite um alerta no navegador onde retorna o valor FALSE ou TRUE.

Função "prompt()": Emite um alerta no navegador solicitando usuário que digite um valor, o qual vai ser atribuido a Função.

Strings

Podemos usar a barra invertida (\) para escrever String "Especiais"

INDICE DE UMA STRING

As Strings tem indices, ou seja, cada letra tem um indice o qaul recebe um valor e esse valor sempre começa em 0;

Exemplo string: nome = "Moises"

Função = console.log(nome,[0]) "vai retornar somente "M".

BUSCAR INDICE DE UMA STRING

Exemplo 1 String: nome = "Moises Lopes"

Função = console.log(nome.indexOf("Lopes")) vai retornar o indice onde inicia a palavra "Lopes".

Exemplo 2 String: nome = "Moises Lopes"

Função = console.log(nome.indexOf("Lopes", 2)) vai retornar o indice onde inicia a palavra "Lopes" a partir do indice 2.

Exemplo 3 String: nome = "Moises"

Função = console.log(nome.lastIndexOf("M")) vai retornar o indice onde está "M" a partir do ultimo indice.

PRESSÕES REGULARES DE UMA STRING

Exemplo 3 String: nome = "Moises"

Função = console.log(nome.match(/[a-z]/g)) Retorna todas as letras Minusculas da String .

Exemplo 3 String: nome = "Moises"

Função = console.log(nome.search(/M/)) Retorna o indice do que está sendo buscado .

Exemplo 3 String: nome = "Moises"

Função = console.log(nome.replace("Moises","Lopes")) Substitui uma palavra pela outra.

Exemplo 3 String: nome = "Moises"

Função = console.log(nome.slice(1,3)) Retorna de um indice a outro [ "oi"].

Números negativos vão pegar os indices do final para o início da String.

TAMANHO DE UMA STRING

Exemplo 3 String: nome = "Moises"

Função = console.log(nome.length) Retorna quantidade de caracteres da String.

DIVISÃO DE UMA STRING

Exemplo 3 String: nome = "Moises Evangelista Lopes"

Função = console.log(nome.split(" ")) Retorna as palavras de ums String, sem o Espaço (" "), dentro de uma Array.

DEIXAR STRING TODA MAIUSCULA OU MINUSCULA

Exemplo 3 String: nome = "Moises Evangelista Lopes"

Função = console.log(nome.toUpperCase()) Retorna a String com todos os caracteres maiusculos.

Exemplo 3 String: nome = "Moises Evangelista Lopes"

Função = console.log(nome.toLowerCase()) Retorna a String com todos os caracteres maiusculos.

Numbers

CONVERÇÃO DE NUMBER PARA STRING

Função: " console.log(num1.toString() + num2 ) "

OBS.: Fazendo desta forma, o valor da variável não é alterado.

CONVERÇÃO DE NUMBER PARA BINÁRIO

Função: " console.log(num1.toString(2) ) "

OBS.: Mostra a representação Binária, sem perder o valor da variavel.

ARREDONDAMENTO DE CASAS DECIMAIS

Função: " console.log(num1.toFixed(2) ) "

OBS.: Fixa o número em 2 casas decimais, sem perder o número da variavel.

VERIFICANDO SE UM NUMBER É INTEIRO

Função: " console.log(Number.isInteger(num1) ) "

OBS.: Verifica se o número é inteiro e retorna os valores TRUE ou FALSE.

VERIFICANDO SE UM NUMBER É "NaN" ( Not a Number )

Função: " console.log(Number.isNaN(num1) ) "

OBS.: Verifica se o número é NaN e retorna os valores TRUE ou FALSE.

IMPRECISÃO DOS NÚMEROS COM CASAS DECIMAIS

OBS.: Uma forma de contornar isso, é não fazendo calculos com números decimais.

Exemplo: Calculo " 0.7 + 0.1 " = ((0.7*100)+(0.1*100))/100

OBJETO "MATH"

Arredondamento para baixo: " Math.floor(num1) "

Arredondamento para cima: " Math.ceil(num1) "

Arredondamento automatico: " Math.round(num1) "

Maior número de uma sequencia: " Math.max(num1,num2,num3...) "

Menor número de uma sequencia: " Math.min(num1,num2,num3...) "

Numeros Aleatórios entre 0 e 1: " Math.random() "

Potência: " Math.pow(2,10) " é igual a 2 elevado a 10.

Arredondamento automatico: " Math.round(num1) "

Arrays

Arrays são listas de coisas ( Nomes, Numeros, Obejtos, e etc )

ACESSAR INDICE DE UM ARRAY

" console.log(array[0]) "

TAMANHO DE UM ARRAY

" console.log(array.length) "

ADICIONANDO ITEM NO FINAL DO ARRAY

" array.push("Objeto") "

ADICIONANDO ITEM NO INICIO DO ARRAY

" array.unshift("Objeto") "

REMOVENDO ITEM DO FINAL DO ARRAY

" array.pop() "

REMOVENDO ITEM DO INICIO DO ARRAY

" array.shift() "

DELETANDO ITEM DO ARRAY

" delete array[0] "

Desta forma, o valor do indice que teve um item excluido, fica com valor de "vazio".

PEGAR UM RANGE NO ARRAY

" array.slice(1,2) "

Desta forma, retornar os valores do indice 1 até o 2.

" array.slice(1, -1) "

Desta forma, retornar os valores do indice 1 até o penultimo item do array.

VERIFICANDO SE O OBJETO É ARRAY

" array instanceof Array "

Retorna TRUE se for um Array, e FALSE caso não seja.

REMOVENDO ITEM DO INICIO DO ARRAY

" array.shift() "

OBS.: Podemos usar CONST para definir Arrays, pois o que alterado são seus valores internos e não a constante "Array".

Funções

Funções executam ações dentro do código.

A nomenclatura para as funções segue a mesma regra de nomenclatura das variáveis.

DECLATANDO UMA FUNÇÃO

" function nomeFunção ( valores de Input ) { o que a função vai fazer } "

RETORNANDO UM VALOR COM A FUNÇÃO

" function nomeFunção ( varDeInput ) { return "varDeInput" } "

EXEMPLOS DE FUNÇÃO

Declaração de Função Padrão

function soma(x,y) {
const resultado = x + y;
return resultado; }

Declaração de Função dentro de uma CONST

const raiz = function soma(n) {
return n ** 0.5; };

Declaração de Função Arow

const raiz = (n) => n ** 0.5

Objetos

São grupos de dados que são relacionados entre si.

Declaração de Objeto

const objetoPessoa1 = {
nome: 'Moises',
sobrenome: 'Lopes',
idade: 55 }


const objetoPessoa2 = {
nome: 'Nome',
sobrenome: 'Sobrenome',
idade: 99 }

Função que Cria Objetos

function criaPessoa (nome, sobrenome, idade) {
return {
nome: nome,
sobrenome: sobrenome,
idade: idade
}
}
const pessoa1 = criaPessoa ('Moises', 'Lopes', 26)

Lógica de Programação

OPERADORES DE COMPARAÇÃO

  • Atribuição ( = )

  • Maior que ( > )

  • Maior ou Igual ( >= )

  • Menor que ( < )

  • Menor ou Igual ( <= )

  • Igualdade ( == )

    Geralmente NÃO É USADO

  • Igualdade Estrita ( === )

    Verifica valor e tipo

  • Diferente ( != )

    Geralmente NÃO É USADO

  • Diferente Estrito ( !== )

    Verifica valor e tipo

OPERADORES LÓGICOS

  • E - AND ( && )

  • OU - OR ( || )

  • NÃO - NOT ( ! )

VALORES QUE RETORNAM "FALSE" e "TRUE"

  • FALSE

  • 0

  • " " - String Vazia

  • Null

  • Undefined

  • NaN

  • OBS.: TUDO QUE FOR DIFERENTE DE TODOS OS DADOS ACIMA IRÃO RETORNAR "TRUE"

"IF", "ELSE IF" E "ELSE"

  • IF ( condição ) { código }

    Se for TRUE excecuta o código, se for FALSE não executa nada.

  • IF ( condição ) { código } ELSE IF ( condição ) { código }

    Se for TRUE excecuta o IF, se for FALSE o ELSE

  • ELSE = Se tiver varias verificações, usa ELSE IF, se tiver só uma, usa ELSE
